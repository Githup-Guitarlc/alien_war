#双向分支
  if...else...语句
    if 条件表达式:
      语句1
      语句2
      语句3
      ...
    else :
      语句1
      语句2
      语句3
      ...
 双向分支有两个分支，当程序执行到if...else...语句的时候，一定会执行if或else中的一个,也仅仅只会执行一个
 缩进问题，if 和 else 一个层级，其余语句一个层级
